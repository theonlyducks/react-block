## Getting started

### Requirements

```shell
yarn add react react-dom
```

### Installation

```shell
yarn add @theonlyducks/react-block
```

### Usage

#### Example

```jsx
import '@theonlyducks/react-block/dist/style.css';
import { BlockDuck } from "@theonlyducks/react-block";

<BlockDuck
    opacity={5}
    blocking={true}
>
    Hi!
</BlockDuck>
```

## Development

Install
```shell
yarn
```

Start server listening https://localhost:3000
```shell
yarn start
```

Run test 
```shell
yarn test
```

Coverage report
```shell
yarn test:coverage
```
